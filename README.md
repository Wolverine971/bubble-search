# Starting the app

- start client
  - cd into the /client and run `pnpm run dev`
- start server
  - cd into the /server and start the python server with `pnpm run start-spacy-server`
  - cd into /server and run `pnpm run dev`

## Needed to run

- chatgpt api keys
- supabase
